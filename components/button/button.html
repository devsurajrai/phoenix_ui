All Floating buttons are also available in four sizes which are Small, Medium ,Large and Extra Large. These sizes can be achieved by adding the classes btn-sm, btn-md, btn-l, btn-xl respectively. Remember you'll need to set the parent's position to relative to make these buttons float according to that parent.
Remember: for the demo purpose "position: absolute" is used for the buttons but you need to change that to "position:fixed"
